2.3 Code Snippets

We have two types of code snippets, copyable and non-copyable, both require the addition of extra JS placed at the top and bottom of this file.

/**@brief A copyable code snippet*/
void main(void)
{
	while(1)
	{
		/* TODO: Write Code*/
	}
}
/**@brief A non-copyable code snippet*/
void main(void)
{
	while(1)
	{
		/* TODO: Write Code*/
	}
}
