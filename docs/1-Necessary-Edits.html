1.3 Do Not Touch

Each HTML page contains header and footer data which forms part of the automation of the templating.
Below I will list the lines and their function, so the author can determine whether they are needed or not. 

  • <meta charset="utf-8">
    Specifies character encoding.
  • <meta name="viewport" content="width=device-width, initial-scale=1.0">
    Supports different viewports.
  • <link href="css/renesas_layout_style.css" rel="stylesheet" type="text/css">
    Specifies the CSS template to use.
  • <script type="text/javascript" src="js/lab-templater.js"></script>
    Specifies the JS templating script to use.
  • <link rel="stylesheet" href="js/highlight/styles/atom-one-light.min.css">
    Specifies the syntax highlighting stylesheet for code snippets.
  • <script type="text/javascript" src="js/highlight/highlight.min.js"></script>
    Specifies the syntax highlighting javascript for code snippets.
  • <script>hljs.highlightAll();</script>
    Function to scan document and perform syntax highlighting where applicable.
  • <script>lab_templater.Generate_header();</script>
    Function to generate templated banner (with side navigation menu).
  • <script>lab_templater.Prepare_modal();</script>
    Calls the templater function to scan the document and apply the modal thumbnails class to specified images.
  • <script>lab_templater.Generate_next_page_button("Next...");</script>
    Calls the templater function to generate the next page button with button text specified in argument.
  • <script>Setup_code_copy();</script>
    Calls the templater function to enable and place code copying buttons in code snippets where applicable.
